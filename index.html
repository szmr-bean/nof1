A better benchmark
Not just “predict the next candle”. Evaluate behavior under constraints.

这个页面用“观赛式”信息架构展示一个交易基准：左侧是总资产曲线，右侧是基准说明与参与者列表。 这类布局在 nof1.ai 的公开截图中也很常见：曲线区域 + 说明文区并列呈现。

  • 统一输入：相同初始资金与数据频率。
  • 统一约束：持仓上限、最大回撤、换手率预算。
  • 可复现：每一次“决策快照”都可追溯。

你可以把 app.js 里的 tick() 替换成 WebSocket / SSE 拉取真实数据，即可变成真正的“直播榜单”。

About

What you get
A deployable static site
  • 纯前端（HTML/CSS/JS），无需构建步骤。
  • Chart.js 通过 CDN 引入，开箱即用。
  • 数据层使用可替换的 dataAdapter，支持 REST / SSE / WebSocket。
  • 默认暗色美学：高对比、细边框、密排信息。
Deploy
Netlify / Vercel / GitHub Pages

把整个文件夹当作静态站点发布即可：

  1. 本地预览：npx serve 或任意静态服务器。
  2. GitHub Pages：把文件放到仓库根目录或 /docs。
  3. Vercel：选择 “Other / Static”，无需构建命令。

详细步骤见 README.md。
